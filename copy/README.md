## How to make changes
After you click on a page, the editor will open.

Once you're done editing, press the green **Commit changes** button to save your edits.

## How to edit markdown
Unless you want specific formatting changes, you should just be able to find the text that's currently on the site and replace it with the changes. For larger and more in-depth edits, refer to the [Markdown Guide](https://www.markdownguide.org/cheat-sheet/).
